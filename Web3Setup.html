🦊 ربط المحفظة — MetaMask Wallet
Connect your crypto wallet to identify as a Guardian

الخطوة 1 من 3 — ثبّت MetaMask

MetaMask هي محفظة رقمية مجانية تعمل كإضافة في المتصفح. بدونها لا يمكن التعامل مع البلوكشين.
MetaMask is a free digital wallet browser extension. Required for all blockchain interactions.

🦊 تحميل MetaMask مجاناً
الخطوة 2 من 3 — اتصل بالمحفظة

بعد تثبيت MetaMask، اضغط الزر أدناه لربط محفظتك بالمشروع:

0x...
Balance: — ETH
أضف هذا الكود في Index.html

انسخ هذا الكود وضعه قبل إغلاق </body> في أي صفحة تريد ربط المحفظة فيها:

<!-- MetaMask Wallet Connection --> <script src="https://cdn.ethers.io/lib/ethers-5.7.umd.min.js"></script> <script> async function connectWallet() { if (!window.ethereum) { alert('ثبّت MetaMask أولاً / Install MetaMask first'); window.open('https://metamask.io', '_blank'); return; } try { const accounts = await window.ethereum.request({ method: 'eth_requestAccounts' }); const addr = accounts[0]; // Save wallet address localStorage.setItem('fl_wallet', addr); // Show short address const short = addr.slice(0,6) + '...' + addr.slice(-4); document.getElementById('walletBtn').textContent = '🦊 ' + short; document.getElementById('walletBtn').style.background = 'var(--green)'; console.log('Connected:', addr); } catch (err) { console.error(err); } } // Auto-reconnect if previously connected window.addEventListener('load', () => { const saved = localStorage.getItem('fl_wallet'); if (saved && window.ethereum) { const short = saved.slice(0,6) + '...' + saved.slice(-4); const btn = document.getElementById('walletBtn'); if (btn) { btn.textContent = '🦊 ' + short; btn.style.background = 'var(--green)'; } } }); </script> <!-- Add this button wherever you want --> <button id="walletBtn" onclick="connectWallet()" style="background:var(--red);color:#fff;border:none; padding:10px 20px;cursor:pointer;font-size:.8rem;"> 🦊 اتصل بالمحفظة </button>

✓ مجاني تماماً. ربط المحفظة لا يكلف شيئاً. فقط تثبيت الإضافة.

📡 بث لامركزي — Livepeer Streaming
Replace browser camera with decentralized Livepeer stream
▼
الخطوة 1 — احصل على API Key مجاني

1. اذهب إلى livepeer.studio
2. أنشئ حساباً مجانياً
3. اذهب إلى Settings → API Keys
4. اضغط "Create API Key"
5. انسخ الـ Key

أضف هذا الكود في live.html — استبدل الكاميرا بـ Livepeer
// 1. أضف API Key في أعلى ملف live.html var LIVEPEER_API_KEY = window.LIVEPEER_API_KEY || '7f763967-912a-4243-99e9-92505c2bcb9b'; // 2. استبدل دالة startLive بهذه: async function startLive() { var title = document.getElementById('streamTitle').value.trim(); if (!title) { alert('أضف عنواناً للبث'); return; } try { // Create stream on Livepeer var response = await fetch('https://livepeer.studio/api/stream', { method: 'POST', headers: { 'Authorization': 'Bearer ' + LIVEPEER_API_KEY, 'Content-Type': 'application/json' }, body: JSON.stringify({ name: title, profiles: [ { name: '720p', bitrate: 2000000, fps: 30, width: 1280, height: 720 } ] }) }); var stream = await response.json(); // Save stream keys localStorage.setItem('fl_stream_key', stream.streamKey); localStorage.setItem('fl_stream_id', stream.id); localStorage.setItem('fl_playback_id', stream.playbackId); // Show stream details document.getElementById('streamKey').textContent = 'RTMP Key: ' + stream.streamKey; document.getElementById('playbackUrl').textContent = 'Watch: https://livepeer.com/streams/' + stream.playbackId; beginLive(title); } catch(err) { alert('خطأ في الاتصال بـ Livepeer: ' + err.message); } }
⚠ ملاحظة مهمة / IMPORTANT

للبث الحي تحتاج تطبيق RTMP مثل OBS Studio (مجاني). Livepeer يعطيك مفتاح RTMP تضعه في OBS وتبث من هناك. المشاهدون يشاهدون عبر رابط Livepeer اللامركزي.

✓ Livepeer مجاني للبدء. 1000 دقيقة بث + 1000 دقيقة مشاهدة مجاناً شهرياً.

🔐 أرشيف أبدي — Arweave Storage

الخيار الأسهل — Irys (مجاني حتى 100KB)

Irys هو واجهة مبسطة لـ Arweave. الملفات أقل من 100KB مجانية تماماً بدون رسوم.
Irys is the easiest way to upload to Arweave. Files under 100KB are completely free.

🌐 افتح Irys.xyz
اختبر الرفع — Test Upload (محاكاة)

هذا اختبار محلي يوضح كيف يعمل الرفع على Arweave:

📁

اسحب ملفاً أو اضغط للاختبار
Drag or click to test upload

✅ جاهز للرفع على Arweave

أضف هذا الكود في Upload.html
// أضف في أعلى الصفحة: <script src="https://unpkg.com/@irys/sdk/build/esm/web/index.js"></script> // دالة الرفع الحقيقي على Arweave عبر Irys: async function uploadToArweave(file) { try { // Connect Irys with MetaMask const provider = new ethers.providers.Web3Provider(window.ethereum); const irys = new Irys({ url: 'https://node1.irys.xyz', token: 'ethereum', wallet: provider }); // Fund if needed (for files over 100KB) const price = await irys.getPrice(file.size); if (price > 0) await irys.fund(price); // Upload file const tags = [ { name: 'Content-Type', value: file.type }, { name: 'App-Name', value: 'FALASTEEN.INK' }, { name: 'Title', value: file.name } ]; const receipt = await irys.uploadFile(file, { tags }); const permanentUrl = 'https://arweave.net/' + receipt.id; console.log('Uploaded permanently:', permanentUrl); return permanentUrl; } catch (err) { console.error('Upload failed:', err); throw err; } }

✓ الملفات أقل من 100KB مجانية. الصور الصغيرة، الوثائق النصية — مجاناً.
للملفات الكبيرة تحتاج كمية صغيرة من ETH في محفظتك (~$0.01 للـ MB).

⚠ مهم جداً — تشغيل سيرفر محلي

ميزات البلوكشين لا تعمل عند فتح الملفات بـ double click مباشرة.
تحتاج تشغيل سيرفر محلي بسيط. هذا يأخذ 30 ثانية:

# 1. افتح Terminal أو Command Prompt # 2. اذهب لمجلد المشروع: cd /path/to/your/project # 3. شغّل السيرفر (Python مثبت على كل جهاز): python3 -m http.server 8080 # 4. افتح المتصفح على: # http://localhost:8080/Index.html # Windows: python -m http.server 8080

✓ بعد تشغيل السيرفر، افتح http://localhost:8080 في المتصفح. الآن كل ميزات البلوكشين ستعمل.

ملف الإعداد الكامل — config.js

أنشئ ملف config.js في مجلد المشروع وضع فيه مفاتيحك:

// config.js — FALASTEEN.INK Blockchain Config // ضع هذا الملف في نفس مجلد المشروع var FL_CONFIG = { // Livepeer API Key من livepeer.studio LIVEPEER_API_KEY: window.LIVEPEER_API_KEY || '7f763967-912a-4243-99e9-92505c2bcb9b', // Network: 'mainnet' للإنتاج, 'polygon' أرخص NETWORK: 'polygon', // App Name للـ Arweave tags APP_NAME: 'FALASTEEN.INK', // Version VERSION: '1.0.0' }; console.log('FALASTEEN.INK Config Loaded ✓');

ثم أضف في كل صفحة HTML قبل </head>:

<script src="config.js"></script> <script src="https://cdn.ethers.io/lib/ethers-5.7.umd.min.js"></script>
